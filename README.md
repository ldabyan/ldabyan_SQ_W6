# Side Quest Week 6 — Papers Please

## Theme
Papers Please — a Soviet-era border inspection game. The world is reimagined as a gritty checkpoint where the player (a Soviet stamp) must fight off waves of invaders trying to cross the border illegally.

## Setup and Interaction Instructions
Open index.html in Google Chrome using Live Server.
To play via GitHub Pages, visit the link provided on the submission page.

Controls:
- Move: WASD
- Shoot: Spacebar
- Skip to boss: B
- Restart: R
